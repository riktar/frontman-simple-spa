## Used in Frontman

This repo is a starter skeleton made for <a href="https://github.com/riktar/frontman">Frontman</a>

## Getting Started (dev mode)

- Cloning this repo
- `npm i`
- `npm run dev`

## Make production build

- `npm run build`
- `npm run serve`

## Requirement

- Node > v10
- NPM

## Under the hood

- Vite
- React
- Tailwind CSS
- PostCSS
- Recoil
- SWR
